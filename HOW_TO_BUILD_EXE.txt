============================================
PDF FLIPBOOK VIEWER - EXE BUILD GUIDE
============================================

📋 REQUIREMENTS:
- Windows PC (to build Windows .exe)
- Python 3.11+ installed
- All project files

⚙️ STEP 1: INSTALL DEPENDENCIES
Run this command in terminal:
    pip install pillow pygame pyinstaller PyMuPDF

📦 STEP 2: BUILD EXE (Two Methods)

METHOD A - Automatic Build (Recommended):
    python build_exe.py

METHOD B - Use Export Button in App:
    1. Run the flipbook application
    2. Click "Export as exe" button
    3. Build starts automatically

METHOD C - Manual PyInstaller Command:
    pyinstaller --onefile --windowed --name=FlipbookPDFViewer --add-data="sounds;sounds" --exclude-module=matplotlib --exclude-module=numpy --exclude-module=pandas --strip --optimize=2 flipbook.py

✅ STEP 3: FIND YOUR EXE
- Location: dist/FlipbookPDFViewer.exe
- Size: 20-30 MB (includes PDF library)

🎯 FEATURES:
✓ Desktop application (NO webview) - Pure tkinter GUI
✓ PDF support with page thumbnails sidebar
✓ Realistic page curl animation effect
✓ Page navigation (Previous/Next buttons)
✓ Zoom In/Out controls
✓ Fullscreen mode (press Esc to exit)
✓ Download PDF functionality
✓ Export to exe button (build from within app)
✓ Sound effects (page turn sounds)
✓ Professional UI matching reference screenshots

📝 USAGE:
1. Run FlipbookPDFViewer.exe
2. Click "Load PDF" button
3. Select a PDF file
4. View with thumbnails on left sidebar
5. Use Previous/Next buttons to navigate
6. Click thumbnails to jump to specific page
7. Use Zoom +/- to adjust view
8. Click Fullscreen for full screen viewing
9. Click Download to save PDF copy
10. Click "Export as exe" to rebuild executable

🔧 REDUCE FILE SIZE TIPS:
1. PyMuPDF is required for PDF support (~15MB)
2. Exclude more modules if not using certain features
3. Compress sound files to lower bitrate
4. Use --upx-dir flag with UPX compressor (advanced)

💡 CUSTOMIZATION:
- Replace sounds in 'sounds/' folder with your own .wav files
- Change colors in flipbook.py (search for hex color codes)
  * Background: '#8B9DA8'
  * Sidebar: '#2C3E50'
  * Controls: '#34495E'
  * Top bar: '#6C7A89'
- Modify window size at line: self.root.geometry("1400x800")
- Adjust page curl effect in add_page_curl_effect() method

📁 PROJECT STRUCTURE:
flipbook.py              - Main PDF viewer application
build_exe.py             - Automatic EXE builder
create_sample_pdf.py     - Sample PDF generator
sample_flipbook.pdf      - Demo PDF file
sounds/                  - Sound effects folder
  ├── page_turn.wav      - Page navigation sound
  └── play.wav           - Animation start sound

🚀 DEPLOYMENT:
The generated .exe is standalone - no Python installation needed on target PC!
Just distribute FlipbookPDFViewer.exe (sounds folder optional for audio).

⚠️ IMPORTANT NOTES:
- Build on Windows to create Windows .exe
- Build on Linux to create Linux executable  
- Replit runs Linux, so download files and build locally on Windows
- Audio works in .exe but not in Replit VNC (no audio device)
- PDF rendering uses PyMuPDF (MuPDF library) - adds ~15MB to exe size
- Page curl effect is simulated with image overlay (lightweight)
